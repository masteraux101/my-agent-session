# 2026 Quantum SDK Benchmarking Project

This repository contains a comparative analysis of the three major quantum programming frameworks as of early 2026: **Qiskit**, **Cirq**, and **Modern Q#**.

## Project Structure
- `/scripts`: Implementation of a Bell State (entanglement) in each framework.
- `/logs`: Execution logs and error reports from local testing.
- `/docs`: Detailed documentation research, GitHub activity reports, and UX analysis.
