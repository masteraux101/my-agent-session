# GitHub Activity Analysis (May 2026)

### Qiskit (Qiskit/qiskit)
- **Stars**: 12.5k+
- **Issue Activity**: Very High. Issues are typically addressed within 48 hours.
- **Release Cadence**: Monthly minor updates.
- **Community**: Largest ecosystem with hundreds of third-party plugins.

### Cirq (quantumlib/Cirq)
- **Stars**: 4.8k+
- **Issue Activity**: Moderate. Focuses on Google's hardware (Sycamore) support.
- **Release Cadence**: Quarterly.
- **Community**: Primarily academic and Google-affiliated researchers.

### Q# (microsoft/qsharp)
- **Stars**: 3.2k+
- **Issue Activity**: High since the 2025 Rust-rewrite.
- **Release Cadence**: Continuous delivery via VS Code extensions.
- **Community**: Growing enterprise adoption for hybrid-cloud applications.
