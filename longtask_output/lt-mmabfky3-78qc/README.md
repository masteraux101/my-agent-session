# Quantum Frameworks Comparison (2026 Outlook)

This repository contains test scripts and a comparative analysis of three major quantum computing frameworks: **Qiskit**, **Cirq**, and **Modern Q#**.

## Repository Structure
- `scripts/`: Implementation of a Bell State generation in each framework.
- `logs/`: Execution output and simulated error logs.
- `docs/`: In-depth research, GitHub metrics, and the final analysis report.
