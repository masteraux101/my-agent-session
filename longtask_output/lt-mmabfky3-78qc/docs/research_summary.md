# Quantum Frameworks Research (2026 Context)

## 1. Qiskit (IBM)
- **Latest Version Trend:** Qiskit v1.x/v2.x.
- **Documentation URL:** [https://docs.quantum.ibm.com/](https://docs.quantum.ibm.com/)
- **Focus:** Transitioned to a lean SDK model. High emphasis on the "Serverless" execution model and primitive programs (Sampler/Estimator).

## 2. Cirq (Google)
- **Latest Version Trend:** Cirq v1.4+.
- **Documentation URL:** [https://quantumai.google/cirq](https://quantumai.google/cirq)
- **Focus:** Hardware-specific gate sets, NISQ-era algorithm development, and tight integration with Google's Quantum Engine.

## 3. Q# (Microsoft)
- **Latest Version Trend:** Modern Q# (Azure Quantum Development Kit v1.0+).
- **Documentation URL:** [https://learn.microsoft.com/azure/quantum](https://learn.microsoft.com/azure/quantum)
- **Focus:** Rust-based compiler, web-native execution, and integration with Azure Quantum. Significant performance improvements over the legacy .NET version.
